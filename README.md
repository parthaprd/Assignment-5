1️⃣ What is the difference between var, let, and const?

var is function-scoped, gets hoisted, and can be redeclared.

let is block-scoped, can be reassigned, but can't be redeclared in the same scope.

const is block-scoped and can't be reassigned. Objects and arrays declared with const can still have their contents mutated though const only locks the binding, not the value.

---

2️⃣ What is the spread operator (...)?

It expands an iterable (array, object, string) into individual elements.It looks identical to the rest operator, but rest does the opposite — it collects multiple elements into one.

---

3️⃣ What is the difference between map(), filter(), and forEach()?

map() — loops through the array, returns a new array with each element transformed.
filter() — loops through the array, returns a new array with only elements that pass a test.
forEach() — loops through the array, returns nothing (undefined).

---

4️⃣ What is an arrow function?

Arrow function is a shorter syntax for writing functions using => instead of the function keyword.

---

5️⃣ What are template literals?

Template literals is Strings wrapped in backticks instead of quotes.
